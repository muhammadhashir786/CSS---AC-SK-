defines the foreground color.

foreground meaning?

there are 2 types of things:

1. foreground
2. backgrouund

the thing which is present in the front is called foreground.
the thing which is present behind the front is called backgrouund.
backgrouund is hidden behind the foreground.

Examples of foreground:

text, buttons, icons, link
